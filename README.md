# WTWR (What to Wear?)

A React application that recommends clothing based on the current local weather. The app fetches real-time weather data and filters a wardrobe of saved clothing items to suggest what to wear depending on the temperature.

## Functionality

- Fetches live weather data based on your location using the OpenWeatherMap API
- Displays current temperature and weather conditions with a matching background image
- Filters clothing items by weather type (hot, warm, or cold) so only relevant items are shown
- Allows users to add new clothing items with a name, image URL, and weather type via a modal form
- Form validation shows inline error messages and disables the submit button until all fields are valid
- Clicking a clothing card opens a preview modal with the item's details

## Technologies

- React (Vite)
- CSS Modules
- OpenWeatherMap API
- GitHub Pages
